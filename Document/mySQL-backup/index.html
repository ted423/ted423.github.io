[资料]Linux下备份MySql数据库

发表于 2018-04-30 分类于 Document

Full备份的话可以用mysqldump，这里有资料
https://dev.mysql.com/doc/refman/8.0/en/mysqldump.html

之后可以通过备份Binary Log的方式备份数据库改动。
Binary Log 的运作原理很简单，它只是单纯的将所有会修改到资料库内容的操作记录在 Log 档案中

首先需要启用Binary Log(默认没启用)

修改 MySQL Server 的系统设定档(eg. /etc/my.cnf，最新版貌似改了，需要自己找下)，在 [mysqld] 区块中加上 log-bin=mysql-bin 选项，然后重新启动 MySQL Server。

然后Binary Log会放在/var/lib/mysql
