种子转磁力链接的简单方法以及磁力链转种子的网页

نُشر في 2015-01-01 في Document

种子转磁力链

一个是可以直接用Torrent File Editor

  1. 确认你的文件夹选项中是勾选了“显示隐藏文件”这一项的，无需确认请直接进入下一步
  2. 用下载工具（迅雷）下载种子，会自动在下载目录下生成一个隐藏的种子文件
  3. 将这个种子的文件名复制下来（注意不要复制后缀名……）
  4. 在文件名前面加上磁力链接前缀： magnet:?xt=urn:btih:
  5. 得到磁力链接！

磁力链转种子

严格来说这不是转换而是通过数据库索引来实现的
因此，也不是所有磁力链都能获取到种子（因此希望各位找到种子后上传到这几个服务器，方便后面的人）
