[Terminal]bcdboot创建以及修复Windows引导

  • 2017-08-19
  • 作者 ted423
  • 1347 字
  • 次阅读

在使用这个命令之前，首先需要了解Windows的活动分区的意思。并了解硬盘GPT格式下esp分区的作用，避免因其他原因导致引导失败。

通常来说使用bcdboot c:\windows /s o: /f ALL /l zh-cn即可(其中o:为esp分区盘符，传统MBR格式使用”C:”即可)
