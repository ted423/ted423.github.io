Root

新接触ubuntu（base on debian）的人，大多会因为安装中没有提示 root 密码而不太清楚为什么会这样。
Ubuntu 的默认 root 密码是随机的，即每次开机都有一个新的 root 密码。

如何设定/改变/启用 root 使用者的密码?
sudo passwd root
当你使用完毕后屏蔽 root 帐号 使用:
sudo passwd -l root
这个将锁住 root 帐号.
如何在终端机模式下切换到 root 身份?

这篇文章以前写的，目前建议是sudo或者sudo -i用就好了,新接触Linux者乱进Root反而容易造成混乱 - 2021-07-03

Debian 默认创建的用户不在 sudoers 中，因此大部分用户可能会比较习惯su root - 2021-07-07

snap

最新版的 ubuntu 很多软件都塞到 snap 里了，导致了一个问题就是磁盘占用较高。还有就是会有一定的权限限制。

并且这些软件是 apt 安装的，很多人都是无缝踩坑。

例如看图软件，之前问了才知道是装到了 snap 里。

不过用桌面端的话，个人建议是谨慎清理 snap
