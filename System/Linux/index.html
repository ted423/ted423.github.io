Linux桌面版桌面以及发行版踩坑实录

发表于 2019-06-01 更新于 2022-04-19 分类于 System

本人因能力问题，只能进行肤浅评价。这里只是用作个人记录。希望客观或实用评价的看到这已经可以离开了。

CentOS

  1. 刚装的中文界面，中文系统，输入法没法用，切换直接报错，版本7.6。虽说Linux查资料处理报错是传统，但也没到系统基本功能都要浪费时间的地步吧？
  2. 意图安装Smokeping，查了一圈都是要编译的,各种依赖项。好奇查了下apt-get, apt-get install smokeping，完美符合我这种小白用户的需求。

Rocky

版本8.4 Dell R710找不到硬盘。(据悉是Redhat删了不少驱动)
然后我随手装了Manjaro…….
我是不想装系统折腾半天，又不是不能用(目测Ubuntu也能用)

xfce

没有disk这个程序，看了下kde manjaro里的，似乎需要桌面环境的支持？默认gnome和kde都会带(不过可以手动装)，比较实用，能看硬盘smart和坏扇区。不过目前感觉普通台式机，不接显示器，kde的vnc会比较卡(nomachine也是同样)
(装gnome-disk-utility)
