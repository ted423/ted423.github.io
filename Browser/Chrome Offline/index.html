[转载]Chrome离线安装包获取方法

发表于 2019-07-24 分类于 Browser

来源
https://blog.csdn.net/hailangnet/article/details/51704034

使用这个地址就行
http://www.google.cn/chrome?standalone=1&platform=win64

?standalone=1 指离线安装包，platform=win64 指64位Windows 版本。

相对更喜欢离线安装包，会装到”C:\Program Files (x86)”,不用每个用户都装一遍

官方文档
https://support.google.com/chrome/answer/95346?co=GENIE.Platform%3DDesktop&hl=zh-Hans#install_win_offline
