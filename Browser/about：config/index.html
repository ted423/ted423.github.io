about:config的一些配置

نُشر في 2016-01-06 في Browser
  • browser.safebrowsing.malware.enabled false可以禁止火狐自动阻止被标为恶意网址的网页
  • devtools.selfxss.count https://ted423.github.io/Browser/no%20allow%20paste/
  • extensions.blocklist.enabled 将设置是否检测阻止的插件，例如旧版flash
  • gfx.color_management.enablev4 改为true后支援ICCV4 （International Color Consortium）
  • permissions.default.image 是否加载图片，值的含义
    1. Allow all images to load, regardless of origin. (Default)
    1. Block all images from loading
    1. Prevent third-party images from loading.
  • browser.search.openintab 新标签页打开搜索栏地址
  • browser.urlbar.trimURLs 设为false后在地址栏显示http://(不然有时候复制网址不会自动加http://)
  • xpinstall.signatures.required 设为false后可关闭关闭“扩展未被验证在Firefox中使用，请谨慎使用”
  • browser.cache.check_doc_frequency 设定检查是否需要更新缓存文档的频率，值的含义
  • 0: Once per session 每个进程一次 每次启动Firefox时检查
  • 1: Each time 每次访问此页时检查
  • 2: Never 不检查
  • 3: When appropriate/automatically 自动
  • plugin.scan.plid.all 是否自动检测系统的插件
    可以将Firefox\browser下新建个plugins文件夹，然后把需要启动的插件放进这个文件夹。
  • browser.tabs.loadDivertedInBackground 后台打开新标签
