去除"欺诈警告:粘贴您不了解的东西时请务必小心，这可能...

发表于 2015-08-08 分类于 Browser

这是一个FF为了想让浏览器显得安全些做的。很神奇的是，却没有”下次不再显示”这个选项(也可能最新版有了？)总之这个设定显示出了FF是多么的不人性化。在FF目前做出很多奇怪决定的状况，我完全不想再继续使用火狐。 -2021-08-18

粘个脚本，被那啥“欺诈警告：粘贴您不了解的东西时请务必小心，这可能会允许攻击者窃取你的身份信息，或者控制您的计算机。请在下方的代码草稿纸中输入”allow pasting”以允许粘贴”烦的要死。
烦的想骂人

具体进到about:config里将devtools.selfxss.count调到100就可以了
