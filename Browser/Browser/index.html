个人浏览器选择对比

发表于 2022-02-06 更新于 2024-07-25 分类于 Browser

Chromium其实已经一统江湖
最终还是跑回去用Chrome了

Firefox

我对FF基本看法就是 哀其不幸，怒其不争

在有这玩意的前提下 no allow paste 去学Chrome的界面，然后学了一手隐藏”http://“的负优化

最早的扩展以及脚本开发都在不断叛逃，然后GM更是自己做了一波死。估计没人理解GM那波作死到底是为了什么。

我始终坚持，软件作为工具，那么最终目的是为了节省别人的时间，而不是浪费别人的时间。
PTPP推送因为某些原因没法用，所以放弃了，Chrome永远的神

Chromium

360极速

目前在用，个人感觉有点烂，放久了会自己崩掉。但起码内核跟的比较紧，基本的一些需求功能都还有。
傻逼地方->下载速度单位M/S
大B小b不分，可想而知程序员水平有多烂

Edge

Edge 主要的问题在收藏移动的操作让人蛋疼(微软自己一直在作死)

Cent

各版本下载http://static.centbrowser.com/

挺喜欢的，但是一个没有收入的闭源软件，能想到后续会如何发展。况且浏览器这块还属于国内软件厂商的兵家必争之地。可惜了
遇到的一个问题
https://www.centbrowser.net/zh-cn/forum.php?mod=viewthread&tid=13972

Opera

内核跟不上，我猜以后应该也跟不上了

Vivaldi

内核倒是跟上了。书签以及快速拨号做的相当不错
缺点就是下载exe非要你确认保留
目前的解决办法是调整组策略，不过会影响整台电脑(试了下还是会弹)
另外随便弹个窗就整个浏览器不让用，这种逻辑太傻逼了
