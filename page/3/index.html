MobaXterm

发表于 2020-08-13 更新于 2023-05-04 分类于 Software

一直想找一个SecureCRT的开源替代品，Tera Term做的不错。
但是其断开自动关闭的傻逼特性实在让人受不了。
虽然我试图用自动记录log来弥补这个问题，但替代手段终究是替代手段。
想想看，你正在跑的脚本因为一时连接不畅断了，你还要翻log找他跑到哪了。
然后保存telnet密码和SSH密码还得用不同的步骤折腾。
所以MobaXterm成了最终解。

另一方面，不要过度迷信开源，毕竟在处理需求上面，商业软件做的更好。

https://mobaxterm.mobatek.net/

缺点

SFTP问题比较大，经常上传下载文件夹会没法保存文件夹结构，所以建议带文件夹结构的还是用 Filezilla 或者 WinSCP
不过这个很多时候是 ssh 断连的问题，因为 sftp 和 shell 是两个独立的 ssh 连接 - 2023-05-03

【迅雷11】是继【迅雷X】之后的换代版PC迅雷客户端。 在【迅雷11】上我们重新设计了主界面框架，许多忠实雷友们过去一直吐槽的太像浏览器的问题，现在已经得到解决。

【迅雷11】的最大亮点，是我们开创性的将【下载】与【云盘】合二为一。

当您下载网络中的文件时，可以灵活的选择将文件下载到【电脑】或者【云盘】。

下载列表

首先要从下载列表说起，在这个版本中，我们将下载列表的任务分类，由原来【迅雷9、迅雷X】采用的纵向排列，改为横向排列。这是由于纵向排列时，每次需要切换任务分类，都要在上下分类之间长距离的移动鼠标，使用较为不便。

悬浮窗

悬浮窗也迎来了变化，从【迅雷7】一直用到【迅雷X】的圆形悬浮窗，现在终于改版啦！

迅雷11新的悬浮窗看上去像是个立方体，当有正在下载的任务时，会在右侧展开一个简短的进度条。

这个全新的设计可以更清晰的显示下载进度，视觉效果也炫酷不少哦。

立即添加（到云盘）

如果您是会员用户，当您选中任意一个下载中的任务时，在下载任务的下方会出现一个新的扩展栏，可以让您快捷的将任务添加到云盘里。

下载列表展示云盘取回

在迅雷11过去的几个版本中，从云盘取回的任务，都只展示在云盘模块的【传输列表】里，对于经常需要查看云盘取回文件的用户来说颇为不便。现在通过下载列表也可以查看云盘取回的文件了。

#更新信息

  • 下载列表的任务分类由纵向排列改为横向排列
  • 采用全新视觉设计的悬浮窗
  • 新增【手机扫码免费加速】功能
  • 左侧栏【下载、云盘】右侧增加数字红点（可展示当前正在下载的任务数，以及正在云添加和上传的文件数）
  • 下载列表中视频任务下方增加【云盘播放】功能
  • 下载列表中非视频任务下方增加【立即添加（到云盘）】功能
  • 下载列表中支持展示从云盘取回的文件
  • 新增【云盘视频智能缓存】功能，改善云盘视频播放体验（可在设置中心停用）
  • 修正某些情况下，播放组件下载解码器失败的问题
  • 云盘增加文件【举报、申诉】功能
  • 云盘传输列表入口移至顶部
  • 云盘回收站改为独立对话框

[问题处理]Windows明明分配了共享权限却无法正常访问网络共享路径

发表于 2023-03-23 分类于 Issue

其实最简单的方法是重启。

说下原因，敲下net use你会看到显示了一堆共享路径的连接。
也就是说Win的话，你不重启他不会重新去连接。
这时候就有可能报错连不上(有可能是缓存，也可能是用之前的用户名密码直接就怼上去了。也别问为啥不弹让输用户密码的框，毕竟微软。)
net use "路径" /del全删掉，大部分重连就正常了。
实在不行还有大杀器net use * /del，不过需要注意这个会删掉网络映射，微软不支持带网络映射的直接重连，需要不删除映射重连的方式大概只有重启。
如果还不行，去凭据管理器里创建相应服务器的用户名密码再重复上述步骤
