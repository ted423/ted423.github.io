[问题处理]Windows 手动安装驱动

Veröffentlicht am 2018-05-02 in Issue

症状

有时候会遇到明明找到了驱动安装包但是Windows一直找不到驱动。

原因分析

微软会有个找驱动的默认路径，一些驱动安装包会自动改掉它，导致微软找驱动不在正确路径

操作

  1. 设备管理器里右键问题设备，点击更新驱动，然后选到第二项，将位置选到“C:\Windows\”并确保“包括子文件夹”已勾选。
  2. 如仍不行，可尝试手动安装，需找到正确的驱动，更新时仍然选到第二项，并点击“从计算机的设备驱动程序列表里选择”，这时要点击从磁盘安装。
  3. 但还有种情况就是提供的驱动程序并不匹配，有次我就强行在inf里添加设备ID安装了驱动，虽然成功了并且能使用，但其实个人并不建议这么做。

etc

微软自动查找驱动的路径保存在HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion下的DevicePath键值里

DiskGenius

这个软件的搜索已丢失分区是非常有用的(不过傲梅分区助手也有这个功能，藏得比较隐蔽，之前没找到)
这个软件严格意义上来讲，我不是很喜欢，分区格式的支持还不如分区助手。有其他分区工具是支持ext4的。

(repaik网站停止运营)

破解的，4.9.2版本看起来是可用来恢复的

Victoria

Victoria 是一款继MHDD无法适应新机器后得最佳硬盘坏道检测工具。
当然，这里必须提下物理坏道和逻辑坏道。
由于以上坏道相关信息来源于英文翻译，并且经常有弄不懂的也发文章，所以概念非常混乱。
通常来说，我的区分方式就是是否能被MHDD用eraser修复，能被修复得就是逻辑坏道，否则就是物理坏道。

但是这个工具有写 [Hint] Recommend 32-bit Windows XP for a best work!
Warning! Windows x64 detected! PIO mode supported on Windows x86 only.

官网:http://hdd.by/victoria/(俄文，建议google翻译英文看…)
下载站:https://download-software.ru/victoria-hdd-download.html

傲梅分区助手

不推荐使用，简单说就是虽然功能看起来非常多，但是很多功能的实现都有问题，样子货
英文官网: https://www.disk-partition.com/
中文官网: http://disktool.cn/
英文版更新更完善，估计国外比较容易赚到钱(现在都一样了)
大眼仔破解: http://www.dayanzai.me/partition-assistant.html

之前一直在用DiskGenius进行磁盘分区调整的相关操作,但是当我因为工作需要将一个XP系统的NTFS无损转FAT32分区的时候，遇见了傲梅分区助手。随后当我发现公司电脑硬盘本身不大，却还分了3个区的时候，又是分区助手的合并分区功能帮了我。后来试图将动态磁盘转为基本磁盘，还是找到了分区助手(虽然我选择了用DiskGenius恢复分区表来还原之前的磁盘，不过DiskGenius中间出了各种问题，吓得我一身冷汗之后才解决倒是)

一方面，这款软件没什么广告而且免费，功能也很全，界面也很干净。老实说，我倒不是特别喜欢界面漂亮的软件，因为很少有软件界面漂亮还在功能上符合我胃口。虽然像什么迁移系统到固态硬盘、启动光盘制作之类的功能让我觉得这款软件还是有点中国软件的风格以外，但还是得说分区助手做的不错。
相对DiskGenius来说，傲梅分区助手不具备恢复文件的功能(不过相对来讲DiskGenius这个功能也是要钱的)。

不过个人感觉这个软件基本是参照着MiniTool Partition Wizard做的，不过功能上还是比不上，很多功能不是很恰当，还有不支持格式化成ext4(英文版支持)。
小问题有点多，对于一个操作硬盘的软件来说这点不大好

DISM是微软本身提供的关于WIM的调整工具，可以用来封装系统镜像。
据软件介绍，DISM++是基于CBS(Component Based Servicing Reference)实现的，可以当作DISM的GUI，但不是DISM 的 GUI版。
就我个人的使用习惯，我是会用这个解封装，但实际操作时会把这软件关掉，用原生DISM命令行做操作，包括开机软件管理我也是用autoruns加载离线系统操作。最后再用这个软件重新封装。
虽然这个软件看起来功能很多，但还是不能涵盖所有的命令
其实我一直以为这软件是开源的，看网上也这么说，今天仔细观察了一下才发现，网站上并没说DISM++开源，也没有开源协议。
