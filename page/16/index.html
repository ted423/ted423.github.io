Windows 10 的各种版本

প্রকাশিত হয়েছে 2020-06-19 সম্পাদিত হয়েছে 2019-05-28 শ্রেনী System

镜像现在一般就两种版本。区别是激活方式不同，其中家庭单语言版基本算是特供版，也就是家庭版上再阉割了一下。专业教育版基于专业版和教育版之间，专业工作站版基于专业版和企业版之间。
教育版功能会比企业版略少一些这样

个人去查了下各个版本的区别，然后还去改了下中文Wiki，然后差点就疯了。同样是微软官方提供的家庭版和专业版的比较，中文和英文的内容还有10%不一样。中文说家庭版有MDM，英文说没有。
还有个 kiosk mode 完全不知道干嘛的(应该是展示模式)，然后中文有个进行测试(???,进行什么测试？)

企业方向的就别说了，最重要的一个SMB Direct(RDMA)中文版干脆就弄没了。

然后不少功能到底支不支持，查都查不到，我估计一些功能微软自己员工都搞不清楚。

言而总之，要是搞不清楚还是直接企业版走起吧，总之这个功能最全不会少。

#Consumer editions(零售版)

家庭版、家庭单语言版、教育版、专业版、专业教育版、专业工作站版。

#Business editions(批量授权版)
专业版、企业版、教育版、专业工作站版、专业教育版。

#N以及KN
Windows 10还特别为欧盟、瑞士提供了“N”（不带媒体播放器）版，以及韩国的“KN”版。这类版本不包含像Windows Media Player或相机这样的媒体相关功能。家用版、专业版以及企业版都有对应的N版或KN版产品。2004年在欧盟的判决下微软被要求发布不绑定Windows Media Player的Windows版本，而这个版本被微软命名为N版。N版与全功能版的售价相同，缺少的功能以“媒体功能包”的形式可以从微软官网上免费下载。

这些版本缺少以下功能：

  • 播放或创建类：
    • 音乐CD回放
    • 数字媒体文件
    • DVD影片光盘
    • 媒体库中的数字内容
  • 创建播放清单
  • 将CD音乐转成数字文件
  • 查阅数字文件中作者和标题的信息
  • 检阅音乐文件中专辑作者
  • 将音乐传入个人音乐播放器
  • 录制回放电视广播
  • Internet Explorer不可回放：
    • 适应流
    • 数字版权管理（DRM）
      缺少的这些功能可以透过Groove音乐和电影与电视这两种应用程序来补足，另外也可安装第三方软件或媒体包以打开媒体文件。

[不再推荐]ultracopier - 复制加强工具

প্রকাশিত হয়েছে 2019-05-27 শ্রেনী Software

目前不建议使用
我测试了2.2.0.5的版本，目前BUG还有点多，不建议日常使用
会有一些Bug,比如回收站里往外拷东西会导致错误的名字，或者权限问题造成无法拷贝的问题。
所以建议一般使用不要开代替，有需要再开
PS: Fastcopy是真的慢，小文件一多连自带的都比过了，大文件的话又完全不需要用

https://ultracopier.first-world.info/
这是我主动寻找的工具，平时在拷贝文件夹的时候，总觉得慢的难以忍受。究其原因，主要是因为小文件太多。
当时就想找一款多线程的复制工具，这样就可以尽量保证跑满硬盘的读写速度。
最初是找到了robocopy,微软自带的工具。它也成功的在复制时把硬盘速度跑满了。
遗憾的是，作为命令行工具，日常操作中，在你打命令行的时候，很多复制都能完成的差不多了。并且蛋疼的是，对一些无法拷贝或者重名文件，命令行工具就是一场灾难。
如果出错了重新复制，他还需要去花时间对比。

然后去找了一些GUI，但实际的效果可能还不如用命令行快。

其中也有试过fastcopy，但感觉实际上这还是单线程复制。CopyHandler（应该是支持多线程，不过没调设置还不如fastcopy,调了之后崩溃了，没再试），做了些复杂操作，进程崩溃了，复制到一半的东西像正常一样显示在硬盘中，试想一下，以后你都搞不清楚文件是不是正常的。

之后找到了ultracopier(开发者告诉我Supercopier 4已经停止支援)，试了下文件夹移动，比fastcopy快了一倍。从软件行为看（用everything看，先把整个目录层次都copy了过去，然后文件创建非常快），应该是多线程。

需要注意的是这个软件是会替换掉微软自带的复制以及移动行为，因此在回收站里操作会有一些问题。还有设置项里performence有个block size,我是调成了4096KB，不然默认的速度会慢，不明白为什么，感觉inode thread是线程的意思，调高似乎会好些

购买专业版是针对CPU有优化的，个人感觉差异不会太大吧？

目前使用发现了一些BUG

  1. 从回收站往外移动时，会把原本的名字丢掉(Windows回收站对名字的特殊处理导致的)
  2. 往exFAT上放文件会容易出问题，经常一取消整个电脑就卡死了
  3. 网络路径也会有些问题，比如网络不稳定的时候
  4. 没有权限的文件夹会比较尴尬，Fastcopy是有提供一个申请管理员权限的按钮
  5. 还有就是剪切时可以粘贴两次到同一目录，第二次点覆盖会导致文件直接丢失
  6. 不支持获取管理员权限，如果一直以管理员权限运行会有些隐患

原理

jpg的内容结束标志为：FF D9 ，其后是否有其他内容jpg算法不关心；而rar的内容开始标志为：52 61 72 21，其前面是否有其他内容rar算法也不关心。因此，将一个jpg和一个rar连接成一个文件，这个文件就既是jpg也是rar了。
