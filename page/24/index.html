由于其配置信息不完整或已损坏，Windows 无法启动该设备

总而言之，这类都属于卸载残留问题。会导致类似问题的还有各类安全软件 - 2021-09-22
光驱没法用了，初步怀疑是daemon tools 安装时选了第2项那个SPd还是啥导致的,有个gear software
当时用的笔记本，自带了光驱，嘛，不过已经是很久以前的事了。
一些其他的设备也可能会出现这个问题，例如一些安全软件，在非常规卸载后会导致一些设备出现这个问题

由于其配置信息(注册表中的)不完整或已损坏，Windows 无法启动这个硬件设备。 (代码19）

  1. 原因分析
    有的时候，由于配置信息(注册表中的)不完整或系统已损坏
  2. 表现症状
    Windows不能启动这个硬件设备，卸载后重新扫描还是显示叹号。
    设备管理器光驱驱动右键属性，提示不正确信息：由于其配置信息(注册表中的)不完整或已损坏，Windows 不能启动这个硬件设备。 (代码 19)
  3. 解决办法
  4. 点开“开始”-运行regedit.exe，进入注册表编辑器,到左边的项目栏里找到
    HKEY_LOCAL_MACHINE\SYSTEM \CurrentControlSet\Control\Class\{4D36E965-E325-11C E-BFC1-08002BE10318}
    选定，在右面窗口找到upperfilter项和loweverfilter项。点右键删除 upperfilter项和loweverfilter项。
    如其他设备则在

HKEY_LOCAL_MACHINE\SYSTEM \CurrentControlSet\Control\Class下搜索upperfilters,但不是所有的upperfilters都可以删除，注意自己分辨。
2. 在设备管理器中卸载有疑问的光驱，卸载之后在点击扫描硬件改动，会找到光驱。这时候光驱就恢复正常了，如果还不可以就重新启动计算机。

参考资料:https://msdn.microsoft.com/en-us/windows/hardware/drivers/install/installing-a-filter-driver
