madVR 瞎写篇

প্রকাশিত হয়েছে 2021-05-12 শ্রেনী Document

参考

http://lysandria1985.blogspot.com/2013/01/3-madvr.html?

devices

projector是投影仪的意思。
properties 这项，madVR本身都提供比较详细的信息了，可以跳过
calibration 是校色，没校色过或低端显示器选disable就好，最下面”disable GPU gamma ramps”如果没有校色过的话就不要勾选
display modes 这里
主要会切换显卡数出分辨率以及刷新率，主要是刷新率符合影片帧率，减少丢帧以及掉帧，需要设备支持，至于是否支持，可以参考Custom display modes
个人建议是使用进入全屏切换，这样平时偶尔看下影片不会有长时间的黑屏
至于Custom display modes，可以用来取代Reclock
原理在这里
http://lysandria1985.blogspot.com/2018/04/3reclock-madvr.html
(但我最后没成功，因为关掉再开后madVR的optimize会重头跑起)

color & gamma
开disable calibration controls for this display的情况下 ，建议pure power curve 预设的2.2就可以，可根据光暗调整，暗环境下，可以增加

HDR
看设备调整，通常未说明显示器都是SDR的，播HDR会有色彩问题，建议直接避开HDR影片，这里的选项也不用管。

processing

去交错不用动，除非你还看DVD。
剩下的除了artifact removal 这块reduce banding artifact 开个low,其他都保持默认就好

scaling algorithms

madVR核心功能

chroma upscaling

建议开NGU-AA，档位看自己选就好，我选high会有奇怪的声音，所以还是medium吧。

image downscaling

SSIM 2D 100% scale in linear light + activate anti-ringing filter
这么设置有个前提，不要拿普通显示器跑4K………

image upscaling

选NGU ，选个very high ，剩下的让madvr decide

upscaling refinement

这里跟锐化有关，个人没调整

rendering

general

delay playback start until render queue is full 勾上
use a separate device for presentation (Vista and newer) 勾上
use a separate device for DXVA processing (Vista and newer) 勾上

smooth motion

打开就好

dithering

最后3个随便选一个就好。

trade quality for performance

这里是降低效果来维持性能的。

下载、资源类软件缺陷记录

প্রকাশিত হয়েছে 2020-06-19 সম্পাদিত হয়েছে 2021-04-24 শ্রেনী Software

BUG

  • aria2c https://ted423.bitcron.com/post/software/zi-yuan-lei/aria2c
  • Transmission
    1. tracker连接性不强，获取tracker信息过慢
    2. 制作种子不支持网络映射
    3. 单线程，牺牲性能降低占用(理论我觉得rt会更好，不过rt不支持改名文件夹这些，并且两个长期不更新的软件，tr反倒是最后更新了一下)
    4. 部分种子识别有问题，部分种子无法使用(我提issue被ban)
  • qbittorrent
  1. 制作的种子dmhy无法读取中文文件名(但应该是网站问题)
  2. 制作的种子tracker中必须隔开一行，否则traker其他软件无法识别
  3. Others https://github.com/qbittorrent/qBittorrent/issues/created_by/ted423
  • 迅雷
  1. 做种相当week,特别是点击添加重新下载时会直接把已下载文件从磁盘抹掉
  2. 现在的版本不能从别的地方下好然后拉进去补充进度了
  3. 迅雷并不会显示具体的tracker信息，并且也不会具体的显示谁给你传数据，也不会显示你在给谁传数据，个人感觉它是努力淡化BT原理，营造一种速度都是迅雷提供的错觉，所以很多迅雷用户体现一种反智现象
  • Deluge
  1. 对中文支持不好,添加种子时路径有中文就出错了，但BT下载的文件夹又可以是中文
  2. 2.0.3大概是不准备也没法支持Windows了

  1. Copy files to a work folder, create exclusion in AV or disable temporarily. I will use d:\work as example.

  2. Install default generic key. Volume key (gVLK) for KMS38 or OEMRET for HWID (see included info TXT files).
    cscript.exe %windir%\system32\slmgr.vbs /ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

    • KMS38 only: Create dummy KMS adress entry using the demo IP range.
      cscript.exe %windir%\system32\slmgr.vbs -skms 192.0.2.69:1833

  3. LTSB 2015 only: rename gatherosstateLTSB15.exe to gatherosstate.exe (rename or overwrite the other).

  4. Depending on process rename respective TargetSKU_XXXX.txt to TargetSKU.txt.

  5. Run gatherosstate.exe by (double-)clicking.

  6. To apply ticket:
    clipup -v -o -altto d:\work
    (note the trailing )

    • HWID only: force activation with:
      cscript.exe %windir%\system32\slmgr.vbs /ato
