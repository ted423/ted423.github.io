记一次Raid卡固件升级(刷IT mode)

主要步骤
rufus写Freedos到USB盘

Copy the following three files from your download to our bootable USB stick.

  • sasbios_rel/mptsas2.rom
  • Firmware/HBA_9211_8i_IT/2118it.bin
  • sas2flsh_dos_rel/sas2flsh.exe

这些文件都放在根目录(不保留文件夹)

需要注意的是broadcom里搜索比较坑，不要分词搜，要一个字符串搜，例如”9211_8i_Package_P20”,否则你就会搜不到东西

bit Byte bibyte(雾) 存储单位，速度单位的一些说明

发表于 2022-03-27 分类于 Document

在计算机中 b=bit B=Byte
其中 1Byte=8bit
B=Byte属于习惯用法，并不表示byte在使用中必须大写。
但是在计算机使用中，KB, MB，GB，大小写必须写对，否则意味这个人技术不过关，基础太烂。
另，MB/s=MBps=8Mbps，通常说的百兆带宽是100Mbps也就是12.5MB/s。
当然百兆带宽属于口语化用词，你不用纠结一个人说千兆带宽下载有100兆（100MB/s）这件事。
当然360浏览器下载速度是M/s的事你也不用纠结，因为说明360浏览器的下载器开发者水平太烂。

这时候冒出来个MiB(Mebibyte)就需要仔细讲讲了
Multiple-byte units
Mebibyte 记作 MiB ，由国际电工委员会（IEC）于1998年制定。这个单位被设计用来某些时候替代MB（megabyte），因为在计算机相关内容中MB有可能被用来等于2的20次方，虽然数值很相近，但MiB与国际单位制（SI）中的MB（10的6次方）还是有严格的区别。
这个单位不常用，不过你知道10TB在Win上只有8.92TB(实际是TiB)就是这个单位做怪了。当然，离不开本博客口头语，微软傻逼。（所以说为啥微软某些地方这么像国内软件啊，不过确实比360强点）

Windows 路由相关

发表于 2022-03-27 分类于 Document

多网络环境可以进行路由优先级调整，避免类似不能访问外网的路由优先级高导致电脑不能上网
其实调整跃点就可以
对于内部的，不跨网段的，其实可以把网关删掉

希望优先级低的，在适配器处，进入ipv4设置，然后点击高级，取消自动跃点，给跃点数填个很高的值

route print
可以看到路由优先级

可以手动指定路由
route add 192.168.0.0 mask 255.255.255.0 192.168.68.254

-p 代表是永久路由

批量重命名软件

发表于 2019-06-20 更新于 2022-03-16 分类于 Software

最早找到的是Total commander让我用的越来越难受，在我看来，这款软件完全没有美感，设计上的，界面上的，虽然功能很强大，设置项也很多，但就像把一大堆东西杂七杂八的融合在一起。

后面找到了

Advanced Renamer

相当不错的工具，但是现在转为了收费软件(个人使用仍然免费)。

http://www.advancedrenamer.com/download

在后面就是寻找替代的开源软件

Ant Renamer

http://antp.be/software/renamer
https://github.com/carry0987/Ant-Renamer
GPLv2
相对来说Ant Renamer的界面以及操作不如Advanced Renamer那么合适可能，但由于其开源，是个相当不错的替代选择。

还有一个小问题就是这个软件大概15年是最后一个版本，是一个很长时间没维护的软件了。

不过目前不影响使用以及功能。

以及

Inviska Rename

https://www.inviska.com/rename/
(2020-05-06打不开，故取消外链)

这个软件的批量方法设计上我觉得不如上面两个，不过他对于音乐的支持相当不错。具体可以看他的文档

SubRenamer

https://github.com/arition/SubRenamer
GPLv3
这个软件可以做到对应命名，在命名不是很规则的时候很好用，比如带hash的文件名。
可以无视文件后缀应该对非字幕也有用。(只试了字幕那块可以加载一堆txt批量重命名，视频那块加载其他后缀没试，虽然感觉没啥问题)

PowerRename

MIT
PowerToys 的一个组件，啥都很强，看起来也很好
就是枚举的实现有些智障。
严格来说不是很推荐

Bulk Rename Utility

https://www.bulkrenameutility.co.uk/
个人免费
看起来界面很丰富，但感觉不太好用，也没法枚举,不推荐
