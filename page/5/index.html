WinRAR是一款强大的压缩文件管理器，它提供了RAR和ZIP文件的完整支持，能解压ARJ、CAB、LZH、ACE、TAR、GZ、UUE、BZ2、JAR、ISO格式文件。

WinRAR 从 4.20开始更好地利用了多处理器核心，不过为了加快压缩速度增加了内存占用。另外，RAR常规解压算法的速度稍有提高，恢复卷的创建和处理可以使用多个CPU核心来获得更高的处理速度。

更新日志

https://www.rarlab.com/rarnew.htm

官网

http://www.rarlab.com/

为什么不用7-zip

1. 还在用VC6编译(现在可以选择7-Zip-zstd)
2. 设置项由于没有UAC权限会直接报错，但MPC-HC对待这种问题已经有解决方案
3. 相较于Winrar,在压缩包中无法运行非单文件的程序(由于只会解压单个文件)
4. 不能批量把文件夹压成单个压缩包

缺点

7-zip支持打开的压缩格式更多,支持的压缩方式更多

[问题处理]update-initramfs failed

发表于 2023-06-07 更新于 2022-05-23 分类于 Issue

Ubuntu apt upgrade 时报错

其实是个老问题了，每次 apt upgrade 都会报错，实际就是 /boot 分区分的太小，然后塞不进了就报错

当时参考https://superuser.com/questions/1601194/update-initramfs-error-24-write-error-cannot-write-compressed-block解决

先去 /boot 分区里看下老版本是啥
然后搜下”linux-image” 或者

理论卸掉一个 sudo apt remove linux-image-5.4.0-109-generic 就能一切继续了

你也可以一口气把老的都卸掉

What's the difference between an image, photo, and a picture 的区别

发表于 2023-06-06 更新于 2022-05-19 分类于 Document

来源: https://www.computerhope.com/issues/ch000169.htm

Digital images, photographs, and pictures
Image - Any visual object that’s modified or altered by a computer or an imaginary object created using a computer.
Photo or photograph - Anything taken by a camera, digital camera, or photocopier.
Picture - A drawing, painting, or artwork created on a computer. A picture is also used to describe anything created using a camera or scanner.

Digital images, photographs, and pictures
Image - 由计算机修改或创建的视觉对象或虚构对象。
Photo or photograph - 由照相机、数码相机或复印机拍摄的图像西。
Picture - 在计算机上创建的绘图、绘画或艺术品。 Picture还用于描述使用相机或扫描仪创建的图像。理论上Picture包含Image以及Photo

音乐播放相关

  • Aimp: http://www.aimp.ru/index.php?do=download
  • foobar2000: http://blog.sina.com.cn/go2spa|官网
  • Autolyric: http://www.autolyric.com/ （唯一支持多行输出歌词的配合专门的 </tags/多行(Autolyric)> ,相当有Feel，网站又挂了，笑）

视频播放器类

  • MPC-HC: http://mpc-hc.org/|clsid2’s Github(GPLv3，官网停更了，Github上还有开发者在维护)
  • MPC-BE: http://mpcbe.sourceforge.net/(GPLv3)(这是一个你需要追更Nightly的软件)
