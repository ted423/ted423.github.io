Sumatra PDF是由Krzysztof Kowalczyk所开发的轻量级PDF查看器，属于自由软件。使用GPLv3协议

Sumatra PDF 由下列两个函式库组成：

MuPDF - PDF函式库
Poppler - PDF函式库

Sumatra PDF的设计风格极简。软件的安装档案体积小，只有大约1MB，比起Adobe Reader的安装档案27.5MB小很多，启动也较快速，并且拥有多20多种语言接口。由于只有一个档案，可以视为是可携式软件。

Sumatra PDF有一个实用的功能，就是它可以记住每一个PDF档案上使用者最后浏览的页面，这使得浏览电子书非常方便。

SumatraPDF, also known as Sumatra, is a PDF reader for Microsoft Windows only. The program also opens Open XML Paper Specification, DjVu, EPUB, XPS, CHM, CBZ and CBR, and MOBI files. It is written primarily by two contributors: zeniko and Krzysztof Kowalczyk

Sumatra includes free software code under version three of the GNU General Public License, while the included Unrar utility is under a commercial “all rights reserved” freeware licence, making Sumatra overall freeware.

madVR 瞎写篇

发表于 2021-05-12 分类于 Document

参考

http://lysandria1985.blogspot.com/2013/01/3-madvr.html?

devices

projector是投影仪的意思。
properties 这项，madVR本身都提供比较详细的信息了，可以跳过
calibration 是校色，没校色过或低端显示器选disable就好，最下面”disable GPU gamma ramps”如果没有校色过的话就不要勾选
display modes 这里
主要会切换显卡数出分辨率以及刷新率，主要是刷新率符合影片帧率，减少丢帧以及掉帧，需要设备支持，至于是否支持，可以参考Custom display modes
个人建议是使用进入全屏切换，这样平时偶尔看下影片不会有长时间的黑屏
至于Custom display modes，可以用来取代Reclock
原理在这里
http://lysandria1985.blogspot.com/2018/04/3reclock-madvr.html
(但我最后没成功，因为关掉再开后madVR的optimize会重头跑起)

color & gamma
开disable calibration controls for this display的情况下 ，建议pure power curve 预设的2.2就可以，可根据光暗调整，暗环境下，可以增加

HDR
看设备调整，通常未说明显示器都是SDR的，播HDR会有色彩问题，建议直接避开HDR影片，这里的选项也不用管。

processing

去交错不用动，除非你还看DVD。
剩下的除了artifact removal 这块reduce banding artifact 开个low,其他都保持默认就好

scaling algorithms

madVR核心功能

chroma upscaling

建议开NGU-AA，档位看自己选就好，我选high会有奇怪的声音，所以还是medium吧。

image downscaling

SSIM 2D 100% scale in linear light + activate anti-ringing filter
这么设置有个前提，不要拿普通显示器跑4K………

image upscaling

选NGU ，选个very high ，剩下的让madvr decide

upscaling refinement

这里跟锐化有关，个人没调整

rendering

general

delay playback start until render queue is full 勾上
use a separate device for presentation (Vista and newer) 勾上
use a separate device for DXVA processing (Vista and newer) 勾上

smooth motion

打开就好

dithering

最后3个随便选一个就好。

trade quality for performance

这里是降低效果来维持性能的。
