[问题处理]Windows 取消同步中心的启动

প্রকাশিত হয়েছে 2016-08-03 শ্রেনী Issue

问题描述

在访问网络位置的时候，右键点击始终脱机可用会启用同步中心，之后会一直启动，并且无法手动取消同步

解决方案

  1. 在同步中心中找到脱机文件
  2. 进入计算机，找到对应的文件夹，右键取消始终脱机可用的勾选,并删除脱机副本
  3. 找到注册表HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Syncmgr\Handlers
  4. 删除{750fdf10-2a26-11d1-a3ea-080036587f03}这一项后同步中心会消失，并且之前设置的同步才会消失
