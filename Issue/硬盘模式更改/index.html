[问题处理]硬盘模式更改(RAID、AHCI、IDE)注意事项，避免蓝屏

发表于 2016-01-14 分类于 Issue

不改注册表直接改硬盘模式会造成蓝屏，因为系统不会加载相应驱动

Widnows 7

改AHCI

找到下面的注册表子项：

在右窗格中，右键单击“名称”列中的“Start”，将值改为 0 (以前的键值是3)
0的值应该是启用的意思，AHCI跟IDE驱动应该能同时启用，如安装的时候就是AHCI模式，改IDE会蓝屏,因为此时的IDE处于未启用状态

改RAID

强烈鄙视HP Z200，硬盘模式只有IDE和RAID+AHCI，结果只有AHCI的驱动还不行

Windows 8 - Windows 10

AHCI则是找到如下注册表

把0的值由3改为0

然后就可以在BIOS里更改为AHCI了，之后等驱动装完后重启几次就好了
IDE则和Win7下没有区别。

RAID还没没试过，所以暂没有
Windows 10似乎在不改注册表的情况下，通过进安全模式来安装驱动。(一台HP Z200里的Win10就是这么操作成功的)
