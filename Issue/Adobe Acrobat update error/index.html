[问题处理]Adobe Acrobat update error

نُشر في 2015-01-01 في Issue

今天升级Adobe Acrobat 一直报错，一开始以为缺少dll是因为运行库的原因后来发现不是。报错信息如下。

Error-1723. —->
“There is a problem with this windows installer package. A DLL required for this install to complete could not be run. Action ISSetupFilesExtract, entry: SFStartupEx, Library- C:\users~\AppData\Local\Temp\MSI649C.tmp “

问题症状是弹出了上述信息框
网上查到了:https://www.facebook.com/adobeacrobat/posts/10151332383682319

大概看了下就是temp的权限问题导致的（可能有重复文件了？）
反正我是用iobit unlocker直接对temp文件夹进行了解锁&删除操作。
然后就能正常安装了
