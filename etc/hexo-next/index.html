发表于 2023-04-23 更新于 2024-08-02 分类于 etc

原标题 为什么试了下 Hexo NexT 然后不用了(现在还是用了)

尽量用简介的语言表达吧，就是tags不会在列表页显示，然后就是auto_excerpt的移除。

前者比较重要，后者的话，会增加很大的维护量，使用hexo-excerpt的话，会对所有文章产生一个阅读全文按钮。

本意是想 Hexo NexT 比较成熟，维护的人比较多，文档也丰富。但博客嘛，能看就行了。

跑回来用这个了，一些原因见 icarus

https://github.com/theme-next/hexo-theme-next/issues/1246
