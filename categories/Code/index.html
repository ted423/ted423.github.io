在 Linux 下如何强制取消挂载(umount) NFS 目录

这种情况下可以考虑lazy umount
即 umount -lf 路径
lazy umount是针对busy而提出的，即可以卸载“busy”的文件系统。
请注意，该方法并不是完全安全的

另外可以参考下https://blog.miniasp.com/post/2008/10/20/How-to-unmount-NFS-directory-under-Linux-when-hanging
不过个人很久之前弄得，不记得当时有没用这个方式
