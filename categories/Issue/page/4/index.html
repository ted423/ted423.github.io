[问题处理]Windows 手动安装驱动

症状

有时候会遇到明明找到了驱动安装包但是Windows一直找不到驱动。

原因分析

微软会有个找驱动的默认路径，一些驱动安装包会自动改掉它，导致微软找驱动不在正确路径

操作

  1. 设备管理器里右键问题设备，点击更新驱动，然后选到第二项，将位置选到“C:\Windows\”并确保“包括子文件夹”已勾选。
  2. 如仍不行，可尝试手动安装，需找到正确的驱动，更新时仍然选到第二项，并点击“从计算机的设备驱动程序列表里选择”，这时要点击从磁盘安装。
  3. 但还有种情况就是提供的驱动程序并不匹配，有次我就强行在inf里添加设备ID安装了驱动，虽然成功了并且能使用，但其实个人并不建议这么做。

etc

微软自动查找驱动的路径保存在HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion下的DevicePath键值里

总而言之，这类都属于卸载残留问题。会导致类似问题的还有各类安全软件 - 2021-09-22
光驱没法用了，初步怀疑是daemon tools 安装时选了第2项那个SPd还是啥导致的,有个gear software
当时用的笔记本，自带了光驱，嘛，不过已经是很久以前的事了。
一些其他的设备也可能会出现这个问题，例如一些安全软件，在非常规卸载后会导致一些设备出现这个问题

由于其配置信息(注册表中的)不完整或已损坏，Windows 无法启动这个硬件设备。 (代码19）

  1. 原因分析
    有的时候，由于配置信息(注册表中的)不完整或系统已损坏
  2. 表现症状
    Windows不能启动这个硬件设备，卸载后重新扫描还是显示叹号。
    设备管理器光驱驱动右键属性，提示不正确信息：由于其配置信息(注册表中的)不完整或已损坏，Windows 不能启动这个硬件设备。 (代码 19)
  3. 解决办法
  4. 点开“开始”-运行regedit.exe，进入注册表编辑器,到左边的项目栏里找到
    HKEY_LOCAL_MACHINE\SYSTEM \CurrentControlSet\Control\Class\{4D36E965-E325-11C E-BFC1-08002BE10318}
    选定，在右面窗口找到upperfilter项和loweverfilter项。点右键删除 upperfilter项和loweverfilter项。
    如其他设备则在

HKEY_LOCAL_MACHINE\SYSTEM \CurrentControlSet\Control\Class下搜索upperfilters,但不是所有的upperfilters都可以删除，注意自己分辨。
2. 在设备管理器中卸载有疑问的光驱，卸载之后在点击扫描硬件改动，会找到光驱。这时候光驱就恢复正常了，如果还不可以就重新启动计算机。

参考资料:https://msdn.microsoft.com/en-us/windows/hardware/drivers/install/installing-a-filter-driver

[问题处理]Windows 取消同步中心的启动

问题描述

在访问网络位置的时候，右键点击始终脱机可用会启用同步中心，之后会一直启动，并且无法手动取消同步

解决方案

  1. 在同步中心中找到脱机文件
  2. 进入计算机，找到对应的文件夹，右键取消始终脱机可用的勾选,并删除脱机副本
  3. 找到注册表HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Syncmgr\Handlers
  4. 删除{750fdf10-2a26-11d1-a3ea-080036587f03}这一项后同步中心会消失，并且之前设置的同步才会消失

[问题处理]Lync联系人名片显示异常

症状

Lync是公司内部使用，公司有自己的Exchange服务器。
联系人资料显示异常，如姓名不对(账号相同，更换其他人使用，但仍显示之前的人的姓名)，电话号码不显示，或者其他。

原因分析

由于删除Lync配置无效，但Lync跟outlook有联系，因此需要找到Outlook里相关配置进行删除。

操作

  1. 退出lync, 打开“%appdata%..\Local\Microsoft\Outlook”
  2. 删除“Offline Address Books”文件夹，重新登录Lync
  3. 如仍有问题，在Outlook里找到联系人名片以及Lync联系人名片删除
  4. 如Outlook里删除不了，可能需要登录网页版的邮件服务器里删除，或者联系Exchange服务器管理员
