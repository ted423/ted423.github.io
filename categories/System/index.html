Windows 10 的各种版本

镜像现在一般就两种版本。区别是激活方式不同，其中家庭单语言版基本算是特供版，也就是家庭版上再阉割了一下。专业教育版基于专业版和教育版之间，专业工作站版基于专业版和企业版之间。
教育版功能会比企业版略少一些这样

个人去查了下各个版本的区别，然后还去改了下中文Wiki，然后差点就疯了。同样是微软官方提供的家庭版和专业版的比较，中文和英文的内容还有10%不一样。中文说家庭版有MDM，英文说没有。
还有个 kiosk mode 完全不知道干嘛的(应该是展示模式)，然后中文有个进行测试(???,进行什么测试？)

企业方向的就别说了，最重要的一个SMB Direct(RDMA)中文版干脆就弄没了。

然后不少功能到底支不支持，查都查不到，我估计一些功能微软自己员工都搞不清楚。

言而总之，要是搞不清楚还是直接企业版走起吧，总之这个功能最全不会少。

#Consumer editions(零售版)

家庭版、家庭单语言版、教育版、专业版、专业教育版、专业工作站版。

#Business editions(批量授权版)
专业版、企业版、教育版、专业工作站版、专业教育版。

#N以及KN
Windows 10还特别为欧盟、瑞士提供了“N”（不带媒体播放器）版，以及韩国的“KN”版。这类版本不包含像Windows Media Player或相机这样的媒体相关功能。家用版、专业版以及企业版都有对应的N版或KN版产品。2004年在欧盟的判决下微软被要求发布不绑定Windows Media Player的Windows版本，而这个版本被微软命名为N版。N版与全功能版的售价相同，缺少的功能以“媒体功能包”的形式可以从微软官网上免费下载。

这些版本缺少以下功能：

  • 播放或创建类：
    • 音乐CD回放
    • 数字媒体文件
    • DVD影片光盘
    • 媒体库中的数字内容
  • 创建播放清单
  • 将CD音乐转成数字文件
  • 查阅数字文件中作者和标题的信息
  • 检阅音乐文件中专辑作者
  • 将音乐传入个人音乐播放器
  • 录制回放电视广播
  • Internet Explorer不可回放：
    • 适应流
    • 数字版权管理（DRM）
      缺少的这些功能可以透过Groove音乐和电影与电视这两种应用程序来补足，另外也可安装第三方软件或媒体包以打开媒体文件。

傻逼微软 记一次关键日志 Microsoft-Windows-Diagnosis-Scheduled 系统维护检测到需要你注意的问题。已将通知发送到“安全和维护”。

突然看到关键日志其实把我吓了一跳。
Microsoft-Windows-Diagnosis-Scheduled 系统维护检测到需要你注意的问题。已将通知发送到“安全和维护”。
想必是电脑快爆炸了。这可把我吓得不行。
然后呢，我的注意力就转移到了这“安全和维护”上。这破玩意到底在哪呢？
我宇宙无敌人间第一人性化操作系统Windows才不会告诉你我把他放哪了，少年啊，去寻找吧。

我找了半天终于在计算机右键属性里找到了这玩意(其实找了两次，第一次找到后气的不行然后关了。结果忘了这东西在哪又找了一次。)
然后发现一点都不”关键”，因为我关了UAC。
电脑是否需要启用UAC我在这里不再做讨论。毕竟Win10系列是你不开UAC通知，但UAC仍然有部分在那里。以至于管理员权限运行的软件读不到网络硬盘。

总而言之我就是折腾了半天发现这是个屁大的小事。（可以设置取消安全和维护对UAC的检测）

[不再推荐]Slax - Linux PE

不推荐的理由在于Ubuntu和Manjaro的安装盘就能做到，并且一般效果比这个还要好
2年未维护，似乎没有提供源码

其实要有个WinPE能看Linux的文件系统也就不用这玩意了
https://www.slax.org

但目前来讲这东西还算有用。
修复了一个ESXi的文件损坏的问题

请注意这个iso不是像别的一样拿什么东西写到移动硬盘或是U盘上，而是直接解压（NTFS文件系统都行）,移动硬盘或U盘中可以保留slax这个文件夹（直接放进去），然后跑boot里的bat。

另外注意这个对移动硬盘可能相性不好，也可能是文件系统的锅，建议优先U盘
