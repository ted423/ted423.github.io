[优化]处理MEGA使用HTML5 File System API占用的空间

Veröffentlicht am 2017-07-29 in Optimization

MEGA的下载是先使用HTML5 File System API先把文件丢到本地，然后再生成一个下载框，但是这些文件是不能通过浏览器自带的清理清掉空间。
蛋疼的在于你下载了多大的文件，Chrome的配置文件就会增加多大的空间，并且越来越大
只能手动找到配置文件夹里的File System(C:\Users\XX\AppData\Local\Google\Chrome\User Data\Default\File System)里手动进行删除
