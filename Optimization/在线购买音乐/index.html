[优化]清除我的文档中右键菜单中的在线购买音乐(聯機購買音樂)

প্রকাশিত হয়েছে 2015-01-01 শ্রেনী Optimization

因为Win10没了WMP，所以这文章应该也没啥用了，不过微软自己也还是喜欢瞎放右键菜单 -2021-08-06
这是Windows Media Player产生的
推荐直接这里处理

取消勾选”启用对此程序的访问”

目錄右鍵菜單中的聯機購買音樂是Windows Media Player產生的，只要在註冊表中刪除下面幾項就行了。

刪除上面幾項下面的WMPAddToPlaylist和WMPPlayAsPlaylist即可。

个人看来只要删掉这项即可 WMPShopMusic（不过我批量处理了add to windows media player和play with windows media player）
不过后面又被WMP给自己加回去了（于是我直接卸载了）。
