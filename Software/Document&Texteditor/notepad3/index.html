[文本编辑器]Notepad3

发表于 2021-07-08 更新于 2021-07-02 分类于 Software
  • https://www.rizonesoft.com/downloads/notepad3/
  • https://github.com/rizonesoft/Notepad3

先说开源协议我没看懂，并没写开源协议的名字，感觉有点像MIT？没细看规则。反正是Copyright类的。
对了下，是3-clause BSD的License

曾经有个notepad2，最大的作用是完全替换微软的notepad，使用的是映像劫持的方式。所以即便装了别的文本编辑器，我也还是喜欢再装一个notepad2以便彻底干死微软的。
可惜弃更了。

所以这个notepad3也能做到，并且开源。就是不知道会不会有Notepad4。

缺点应该就是扩展性方面了，比如简繁转换，格式化代码啥的就没。
