文件恢复类软件

  • 2019-07-24
  • 作者 ted423
  • 907 字
  • 次阅读

之前用过Recuva(不好用，Piriform公司的东西界面不错，但实际都不咋滴，点名批评CCleaner)、Diskgenius(需破解，大部分破解不完全，有些还会崩溃)、Wise的
基本都是帮别人恢复，由于案例不够，除了EaseUS，和上面几个以外，其他的还有待验证。

写在前面，随着SSD的越来越流行，这部分软件大概会逐渐退出历史舞台。

EaseUS Data Recovery Wizard

界面设计相当合理，功能确实非常不错，将别人出问题的SD卡里整个文件夹的东西都还原出来了。
傻瓜化，易操作，效果非常好

官网
https://www.easeus.com/
(有国内官网，但我现在不敢信任经过国内软件商改的东西)

破解版(部分电脑激活有点问题，原因未知)
http://www.dayanzai.me/easeus.html

PhotoRec_CN

开源，免费.........
GPLv2
单看界面没有按文件夹整个恢复，也就是没有目录层级的恢复。就这点来讲不如上面那个。
恢复是一点就全部恢复出来，不会弄个界面让你选

Kickass Undelete

开源，免费
GPLv3
但懒得试了，毕竟SSD大行其道
另外2018年后就没再更新过了

R-Studio

印象里是说非常有名非常强大
http://www.dayanzai.me/r-studio.html

魔方恢复

优点是有个健康度
在SSD上运行并成功恢复出来了，但是健康度100%出来也是坏的(大概因为SSD)

pcmaster

WinFR

https://www.winfr.com.cn/
我印象里我上次试扫不出来东西，这个是 GUI，基于微软官方的功能 https://support.microsoft.com/en-us/windows/windows-file-recovery-61f5b28a-f5b8-3cc2-0f8e-a63cb4e1d4c4

现在发现应该是要先装 Windows File Recovery 并且只能在微软商店安装。

开发傲梅软件助手的公司开发的。

但总之放到列表里

清理了不推荐软件的 url，增加 WinFR- 2024-12-23
