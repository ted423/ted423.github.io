[开源]Ventoy - iso启动器

发表于 2023-06-06 更新于 2020-06-26 分类于 Software

Ventoy是一个专用于启动iso安装盘的启动盘，其本身不会将系统启动盘刻录到U盘/移动硬盘中，而是可以去选择运行指定的ISO。

个人感觉这个软件有些类PE

https://www.ventoy.net/cn/index.html

优势是可以不用反复格式化U盘/移动硬盘，将ISO放入U盘/移动硬盘中即可，也可以支持本地硬盘中的ISO文件，可以一定程度上规避一些Linux启动盘刻录到移动硬盘后无法mount cdrom的问题。可以说兼容性可能比刻录安装盘效果更好些。

不过定位还是有点尴尬，毕竟普通用户很少天天装系统，专业人士，PE，弄安装盘的时间也不会缺
