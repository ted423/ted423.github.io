[软件推荐]影音类软件

发表于 2020-08-21 更新于 2022-05-18 分类于 Software

音乐播放相关

(Aimp建议配合Autolyric;千千静听已死;个人建议如果你不是很了解解码器的话，解码器包是必装的;foobar2000建议下Asion的;另foobar的ESLyric相当棒，不过我还是喜欢Autolyric的多行显示，因为这样能确保有翻译时的感受)

  • Aimp: http://www.aimp.ru/index.php?do=download
  • foobar2000: http://blog.sina.com.cn/go2spa|官网
  • Autolyric: http://www.autolyric.com/ （唯一支持多行输出歌词的配合专门的 </tags/多行(Autolyric)> ,相当有Feel，网站又挂了，笑）

视频播放器类

  • MPC-HC: http://mpc-hc.org/|clsid2’s Github(GPLv3，官网停更了，Github上还有开发者在维护)
  • MPC-BE: http://mpcbe.sourceforge.net/(GPLv3)(这是一个你需要追更Nightly的软件)
  • mpv:https://mpv.srsfckn.biz/(GPLv2,因为放某个视频卡之后我把它放弃掉了)
  • VLC:https://www.videolan.org/vlc/(GPLv2,跨平台，但是官方因为madVR不开源，表示不会支持madVR)
  • Light Alloy:http://light-alloy.verona.im/(毛子的播放器，比potplayer有些差距，秒KM以及国内就没商量了,目前有个不知道怎么看文件信息的问题，OSD点了也不显示)

滤镜&渲染器&解码

  • K-Lite解码器包：http://www.codecguide.com/download_kl.htm
    下面2个都可以通过K-Lite安装
  • LAVFilters:https://github.com/Nevcairiel/LAVFilters/(开源，GPLv2，相较FFmpeg只有分离器和解码的功能)
  • MadVR(最强渲染器)：http://madshi.net/madVR.zip | 更新日志
  • FFmpeg:https://ffmpeg.org/(这是编解码一体的一套工具，开源，LGPL version 2.1，部分组件是GPL的)
  • Reclock(最强音频渲染)：https://forum.slysoft.com/forumdisplay.php?85-ReClock(似乎说可以被MadVR取代了)

工具类

  • Mkvtoolnix(mkv(Matroska)格式制作和处理的工具)： http://www.bunkus.org/videotools/mkvtoolnix/downloads.html#windows
  • MKVExtractGUI(需先装Mkvtoolnix,可提取mkv内挂字幕): http://sourceforge.net/projects/mkvextractgui-2/
  • gMKVExtractGUI(同上，还在维护的版本): https://sourceforge.net/projects/gmkvextractgui/
  • My MP4Box GUI: https://www.videohelp.com/software/My-MP4Box-GUI
    (MP4Box的GUI，13年最后更新，建议重心转到mkv上。
    MP4加字幕后多了一个默认的空字幕轨，建议只做提取用,后面试了ffmpeg命令行照样空字幕轨，笑)
    目前已经不能用了
  • DVD Decrypter:http://www.dvddecrypter.org.uk/(DVD解密以及按章节切割工具)
  • BDinfo: https://github.com/UniqProject/BDInfo(Bluray信息生成)
  • Aegisub: http://www.aegisub.org/(字幕调轴)
  • spek: http://spek.cc/(音乐频谱图获取)
  • CUETools: http://cue.tools/wiki/CUETools_Download(无损音乐整轨切分轨的工具)

压制相关

压制的建议以后看这里 https://github.com/ted423/FXXS-Encode-Guide/blob/main/2.%20Tools/Collection.md

  • UsEa3to(Extract各种轨道，包括原盘，不支持特殊符号及网络路径):https://www.videohelp.com/software/eac3to
  • Megui: https://sourceforge.net/projects/megui/(开源,GPLv2,但我用不来，添加QTGMC各种报错,主要弄不懂)
  • MediaCoder: http://www.mediacoderhq.com/(转码用的)
  • VS: http://www.vapoursynth.com/|Github
  • VSRepoGUI: https://github.com/theChaosCoder/VSRepoGUI(安装VS脚本和升级VS脚本用的，但是网络不稳定的情况下挺难用)
  • Simple-x264-Launcher: https://github.com/lordmulder/Simple-x264-Launcher(写的x264但是有x265)
