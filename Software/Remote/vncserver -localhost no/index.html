vncserver -localhost no

نُشر في 2021-07-19 في Software

记住这条命令，这是我被坑过最惨的一次。(理论上算是我找了个不错的教程？)
第一次能用就没管了，结果重启手动启动连不上。
跑vncserver连不上，看了有端口，也没开防火墙，最后仔细查，结果绑在了127.0.0.1上，本机IP带5901连不通。
默认localhost yes -> [-localhost yes|no] Only accept VNC connections from localhost
这是脑子多大坑才会有这种设置项。
目测是最近出来的？ 不然咋啥教程都不说明
