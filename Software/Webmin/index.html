[开源]Webmin - WebUI for Linux

发表于 2023-07-15 更新于 2023-07-14 分类于 Software

WebUI 是个好文明，其配置项也能方便初学者尽快了解 Linux。
里面的配置项相当丰富，很多甚至我不开配置都不知道还能这么做
前提是不能因此抛弃命令行，否则会成为一个只会点点点的人。如果只想点点点，建议还是Win就好。
https://www.webmin.com/

Virtualmin

强烈不推荐
Virtualmin 是 Webmin 的一个 module，只不过安装和webmin就不一样了，一定要跑脚本，有GPL版和高级版两个版本。我个人不太喜欢。可能因为装了这个以后里面就相当于有了一些广告？

另一方面，里面的功能不太需要，安装使用的是脚本（个人不太喜欢这点），会自动开防火墙并且没有提醒。会自动帮你装邮件的一些软件包（不需要使用）。
这方面的，例如邮件服务器这块，如果使用，我会更倾向于使用Docker

卸载的时候似乎还把我以前 shell 里设的定时任务带走了 —2023-07-14
