[不再推荐][大眼仔~旭]Clover 3.5.4 多标签资源管理器，替代Explorer

发表于 2020-05-27 分类于 Software

不再推荐的原始是买了Directory Opus

目前的版本似乎不少杀软都会报毒，但都是报类似Adware或者Softcnapp这类的，请各位自行判断是否使用。

另外注意需要用管理员模式运行才能保存配置

试过不少的多标签资源管理器这方面的软件，初衷就是找窗口方便，下面资源管理不要叠太多免得一个文件夹打开太多次都不知道，然后看起来还乱。

QTTabbar会影响复制粘贴，没办法只在一个窗口显示，有时会打开新的窗口。Groups会卡死。
其他的不少都没法直接替代资源管理，使用替代方案又有这样那样的问题，譬如迅雷会觉得自己没法开文件夹，又重新开个Explorer。

Directory opus是比较好的，但是因为其破解后会有一些报错，本身又有些贵，所以没用。
相较之下，DO更重型，设置项过多，有些复杂，界面调整有些奇怪的逻辑，主页的显示不会以Windows 默认为主，是自己的一套东西，但个人感觉不如Windows的自带的好。

Clover就比较轻型，也有一点小问题就是了，有时鼠标在标签页上移动，虽然没点击但标签页会粘在上面。而且不像DO可以不开重复的路径

已知问题:如果文件夹内有东西会不断变动，Clover会不断刷新，影响焦点，然后影响使用。
