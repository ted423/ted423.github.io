批量重命名软件类

单独文章

Attribute Changer

同File Property Edit一样可以进行文件属性修改，但主要是通过右键菜单，
http://www.petges.lu/home/download/

File Property Edit

文件属性修改,你可以将普通文件改成系统文件，十分适合用来藏东西
http://www.foryoursoft.com/download.htm

搜索类

Everything

搜索文件用，相当出名的文件搜索工具，会通过主文件表中的文件元数据创建卷上每个文件和目录名称的索引，速度相当快，缺点就是只支持文件名，不支持附加条件的搜索，例如文件大小和创建日期

最新的 alpha 版本看起来会增加很多功能， 包括文件内容搜索 -2024-12-30
http://www.voidtools.com/

FileSeek

这个软件是专门为了搜索文件内容而生的。没有locate32的小毛病，后面感觉用FileLocator会好些，因为我发现FileLocator添加默认为文本的文件类型会比较方便。我印象里FileSeek最早也是直接添加的，不知道是我记错了还是后面改了。
https://www.fileseek.ca/

FileLocator

跟上面算是同质同类的软件
https://www.mythicsoft.com/filelocatorpro/

Copyq

GPLv3

用就是了 之前在用Ditto，不过感觉Ditto小bug有点多。

最主要的是他会导致其他软件卡死。-2024-12-30
https://github.com/hluk/CopyQ

Exeinfo PE

exe信息查询
放在Github上的闭源软件,之前有一堆不同域名的主页

DIE-engine

exe信息查询 开源，比上面那个重，不过支持文件夹安装包和压缩包

HashCheck

Standard 3-Clause BSD License.
https://github.com/gurnec/HashCheck
因为Robin Keir的hash在特殊字符上有些问题，找替代软件找到的。不过只能在右键属性里查看。如进行文件对比，操作量会略微增加。

MyHash

Robin Keir的hash的完全替代品,可惜虽然在Github上，但没有开源。
https://github.com/drag0n-app/MyHash

KeePassXC

强大的密码管理软件以及密码自动填写工具

微软小工具合集

微软的系统工具合集

Registry Workshop

注册表搜索工具,搜索速度快，可以批量操作,不过需要付费
https://hrtsea.com/2133.html
官网：http://www.torchsoft.com/

RightMenuMgr

右键菜单管理，最详尽最专业的，官网已消失，虽然是老软件但在win10下也正常。
https://www.lanzous.com/i39jpri

SpaceSniffer

是一款非常直观的磁盘空间占用查看应用，还支持根据不同类型文件进行过滤显示。后面感觉WizTree更好
http://www.uderzo.it/main_products/space_sniffer/

WizTree

比较而言后者大文件提示会比较直观.百分比占用会比图形更准确，好理解。
前者会边扫图形边变化，前期图形会失真，并且整体速度略慢
据说Wiztree是直接对NTFS文件系统的MFT进行文件分析，不过我测试是对的网络路径(小文件太多估计有影响)
固此，我倾向于用WizTree 完全取代 SpaceSniffer
https://www.diskanalyzer.com/

UnLock IT

现在用 PowerToys了
文件解锁，删除删不掉的文件都可以用
各种解锁软件在我上Win10就开始出问题，比如IObit Unlocker,又比如LockHunter然后我找到了UnLock IT，这是一个免费软件,功能强大。
http://emcosoftware.com/unlock-it/download
