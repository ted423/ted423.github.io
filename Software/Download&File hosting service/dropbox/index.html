下载DropBox离线安装包的方法(可能失效)

نُشر في 2016-12-10

一直无法理解那些给软件下载地址非要给个下载器的。特别对于开代理的，非要忍受那个下载器没速度的奇葩问题。
原来dropbox是提供离线下载包的，现在不知道为什么又取消了。
目前可以在dropbox的(下载页面)[https://www.dropbox.com/install#downloaded]上
找到下载连接：https://www.dropbox.com/download?os=win&full=1
在地址链接后加上“&full=1”即可。
