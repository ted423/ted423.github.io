原文: http://www.weste.net/2007/11-29/21525122417.html
迅雷就在地址前后分别加上“AA”和“ZZ”，然后进行Base64编码

快车是在地址前后分别加上了“[FLASHGET]”，然后Base64编码，编码后有在地址后加上“&id数字&数字”
