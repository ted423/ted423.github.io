Java

http://www.java.com/zh_CN/
Java 8 及之前：Oracle 和 OpenJDK 会提供两个独立的包：
JRE (Java Runtime Environment): 只包含运行 Java 程序所必需的虚拟机（JVM）和核心库。用于部署和运行 Java 应用。
JDK (Java Development Kit): 包含完整的 JRE，外加编译器（javac）、调试器等开发工具。用于开发 Java 应用。
Java 9 及之后：官方（Oracle/OpenJDK）不再单独发布 JRE。现在的做法是，JDK 本身就包含了 JRE。如果你需要一个精简的运行时，可以通过 jlink 工具从 JDK 中创建一个自定义的、最小化的运行时环境。
