不改注册表直接改硬盘模式会造成蓝屏，因为系统不会加载相应驱动。

因为最新的电脑基本不再支持 IDE，所以下面的内容仅供参考，顶多就是一些电脑默认是 RAID 模式可能会有用，不过预装系统一般是 Home 版，通常建议改模式之后重装就好。

系统应该默认不包含 RAID 的驱动（非 OEM 系统）--2025-11-20

Services 里 Start 的 0的值应该是启用的意思，AHCI跟IDE驱动应该能同时启用，如安装的时候就是AHCI模式，改IDE会蓝屏,因为此时的IDE处于未启用状态

Windows 8 - Windows 10

把0的值由3改为0
