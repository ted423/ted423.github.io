[火狐扩展推荐]你不一定知道，但是很有用的扩展

  • 2016-02-16
  • 作者 ted423
  • 1437 字
  • 次阅读

#Add to Search Bar
能比较方便的向搜索列表里加引擎，比火狐自带的识别好用一些
https://firefox.maltekraus.de/extensions/add-to-search-bar

#Copy Link Name
你有试过准备在连接上复制文字但死活选不到正确位置时的痛苦么?
这个扩展就是用来解决这个的。(其实可能找个UC脚本更好一些？)

https://addons.mozilla.org/en-US/firefox/addon/copy-link-name/

需要注意的是用户评论里有The next version (1.3.5.5) contains adware:
并且我注意到火狐默认给装的是 1.3.5.3.1-signed

#DOM Inspector&Element Inspector
Element Inspector:https://addons.mozilla.org/en-US/firefox/addon/element-inspector/
可以查看火狐界面的CSS代码，写userChrome.css,也就是浏览器界面样式必备，一般需配合DOM Inspector

DOM Inspector:https://developer.mozilla.org/en-US/docs/DOM_Inspector
(0.5以后的Element Inspector可以不使用这个,我这调用的是firebug,原生开发者工具貌似没法显示)

#Developer Edition theme enabler
火狐给了Developer Edition一个很酷的外观，正式版却不给用！！！！Developer Edition还天天出问题！！！
于是这个扩展很好地解决了这个问题
https://addons.mozilla.org/zh-CN/firefox/addon/devedition-theme-enabler/

#Flagfox
可以比较清楚的描述网页所在地,不需要设置(我记得还有一个类似的，还有一大堆设置，部分页面还没有显示)
https://addons.mozilla.org/en-US/firefox/addon/flagfox/

#Proxy Selector
https://addons.mozilla.org/zh-CN/firefox/addon/proxy-selector/
能定义不同的代理状态并进行切换，比如全局或者PAC，也可以设置不同的PAC，就不用每次都点到选项里了

#Show parent folder&Go parent folder
能比较清楚的找到书签所在文件夹
https://addons.mozilla.org/zh-cn/firefox/user/white-alice0775/
注意，Show parent folder需要自己勾选列后他才显示

#集图工具(Image Picker)
页面图片批量下载工具，也支持当前窗口打开的所有页面
https://addons.mozilla.org/en-US/firefox/addon/image-picker/

#Url To Qr Code
将电脑打开的网页交由手机打开的最简单的方式
https://addons.mozilla.org/zh-CN/firefox/addon/url-to-qrcode/?src=search
