文本编辑器类的个人选择

本博有俩文本编辑器，其实我用过很多，所以就写写
最早更换文本编辑器主要还是为了代码美化，基本都是基于对javascript的美化效果，试过VS、Sumblime等等等，当时还找了个专门做这类破解的，似乎叫流风清音。
最后发现Everedit不错，价格是100多的终身，我就直接入了正。（可以在很多终端上激活）
小问题也有，例如关联实现有点问题，大概是收益不好，更新不怎么勤快。

之所以写这个是我发现notepad++不支持打开大文件(3g，目测有人把某个文件追加写入到.sh后面了)，然后everedit,notepad3都行。
另外就是nodepad++的扩展，找起来网址都是一堆，但是要找原发布地址就很难，并且很难保证还能用。装起来也比较复杂，有点费时。
不过notepad++支持跨文件替换(就是需要都是打开状态)

notepad3,必装,就是没有扩展之类的功能
sublime text，会有奇奇怪怪的输入法问题(当时写这个似乎是因为在Linux上面用)

[开源]SumatraPDF

由于adobe reader免费，这个软件还是有些尴尬的。不过在epub和其他格式的文件方面很有优势，甚至可以拿来看压缩包形式的漫画。
只不过由于他的名字，不是很显眼。
不过2016到现在都没出新的release,似乎在忙着重构？看epub和cbz使用中都遇到了一些小问题https://forum.sumatrapdfreader.org/t/something-about-epub-and-cbz/2244

官网

https://www.sumatrapdfreader.org/free-pdf-reader.html

github

https://github.com/sumatrapdfreader/sumatrapdf

Sumatra PDF是由Krzysztof Kowalczyk所开发的轻量级PDF查看器，属于自由软件。使用GPLv3协议

Sumatra PDF 由下列两个函式库组成：

MuPDF - PDF函式库
Poppler - PDF函式库

Sumatra PDF的设计风格极简。软件的安装档案体积小，只有大约1MB，比起Adobe Reader的安装档案27.5MB小很多，启动也较快速，并且拥有多20多种语言接口。由于只有一个档案，可以视为是可携式软件。

Sumatra PDF有一个实用的功能，就是它可以记住每一个PDF档案上使用者最后浏览的页面，这使得浏览电子书非常方便。

SumatraPDF, also known as Sumatra, is a PDF reader for Microsoft Windows only. The program also opens Open XML Paper Specification, DjVu, EPUB, XPS, CHM, CBZ and CBR, and MOBI files. It is written primarily by two contributors: zeniko and Krzysztof Kowalczyk

Sumatra includes free software code under version three of the GNU General Public License, while the included Unrar utility is under a commercial “all rights reserved” freeware licence, making Sumatra overall freeware.
