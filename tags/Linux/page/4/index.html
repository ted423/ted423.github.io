Root

这篇文章以前写的，目前建议是sudo或者sudo -i用就好了,新接触Linux者乱进Root反而容易造成混乱 - 2021-07-03

Debian默认创建的用户不在sudoers中，因此大部分用户可能会比较习惯su root - 2021-07-07
原文http://shewolfep.iteye.com/blog/413396

新接触ubuntu（base on debian）的人，大多会因为安装中没有提示root密码而不太清楚为什么会这样。
Ubuntu的默认root密码是随机的，即每次开机都有一个新的root密码。

如何设定/改变/启用 root 使用者的密码?
sudo passwd root
当你使用完毕后屏蔽 root 帐号 使用:
sudo passwd -l root
这个将锁住 root 帐号.

[软件推荐]图片类相关软件

看图

最近没怎么看漫画就比较尴尬，目前还是Honeyview，能用就行，不过感觉他的缩放算法问题很大

我一度是想抛弃ACDsee的，比如文件名长度问题导致的崩溃，又比如某些gif会花。但偏偏他的数据库让人无法舍弃，比方说按宽高搜索图片之类的，速度相当快，相对来讲XnViewMP搜索类别更细，功能性更强，但是Xnview里的结果是一次性的，一旦点开一张图片结果就会消失。就个人感觉，XnView的规划很乱，设计也是乱七八糟，唯有功能比较多而已。
况且XnView和XnViewMP的使用界面和方式还有很多不一样，感觉怪怪的。
MassiGra的各种按键自定义简直可怕，而且支持读取压缩包里的图片，其实SumatraPDF也支持读取压缩包里的图片
强力推荐下Honeyview，给我一种“就是他了”感觉的软件，比MassiGra简单很多，但又能完全取代的一款软件
因为Honeyview说除安全修补，不再更新，所以想找替代，结果Honeyview 2021还有更新，nomacs一年没更新(现在都在action里下最新版)，FastStone MaxView两年。

  • XnViewMP :http://www.xnview.com/en/xnviewmp/ (QT开发跨平台)
  • Honeyview:http://www.bandisoft.com/honeyview/cn/ (商业软件)
  • nomacs: https://nomacs.org/ (GPLv3)
  • ACDsee Ultimate:-|官网
  • MaxView: http://www.dayanzai.me/maxview.html|官网

看压缩包漫画，首选还是Honeyview，SumatraPDF目前感觉用起来不错，nomacs只支持zip/cbz
但如果看对比图或者比较专用的使用的话Honeyview就不太行，滚动图片会移位

图片管理

Xnview

挂个名

Adobe Bridge CC

http://www.adobe.com/cn/products/bridge.html

这款软件是我找来取代ACDsee的图片管理功能的，毕竟ACDsee收费，并且其不能在资源管理器里打开，也不能用其他图片浏览软件打开图片，Adobe Bridge CC 虽然不像ACDsee那样可以根据图片的宽和高来搜索图片，但可以根据图片的长宽比来分类，缺点是界面有点丑。
值得一提的是Adobe lightroom CC 反而不支持通过图片的长宽比来分类，简直让人难以理解。

digiKam

这是一款根植于 Linux 的 KDE 软件，但是跨平台,可以编辑，就是单纯看图不太行，不过可以调用看图软件。目前我的首选。
不过Linux软件讨厌的一点是一不注意就往源目录下”拉屎”,注意不要用他的回收站，不是指的往Windows的回收站丢。
