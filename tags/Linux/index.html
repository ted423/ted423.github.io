[开源]Ventoy - iso启动器

Ventoy是一个专用于启动iso安装盘的启动盘，其本身不会将系统启动盘刻录到U盘/移动硬盘中，而是可以去选择运行指定的ISO。

个人感觉这个软件有些类PE

https://www.ventoy.net/cn/index.html

优势是可以不用反复格式化U盘/移动硬盘，将ISO放入U盘/移动硬盘中即可，也可以支持本地硬盘中的ISO文件，可以一定程度上规避一些Linux启动盘刻录到移动硬盘后无法mount cdrom的问题。可以说兼容性可能比刻录安装盘效果更好些。

不过定位还是有点尴尬，毕竟普通用户很少天天装系统，专业人士，PE，弄安装盘的时间也不会缺

[问题处理]update-initramfs failed

Ubuntu apt upgrade 时报错

其实是个老问题了，每次 apt upgrade 都会报错，实际就是 /boot 分区分的太小，然后塞不进了就报错

当时参考https://superuser.com/questions/1601194/update-initramfs-error-24-write-error-cannot-write-compressed-block解决

先去 /boot 分区里看下老版本是啥
然后搜下”linux-image” 或者

理论卸掉一个 sudo apt remove linux-image-5.4.0-109-generic 就能一切继续了

你也可以一口气把老的都卸掉

在 Linux 下如何强制取消挂载(umount) NFS 目录

这种情况下可以考虑lazy umount
即 umount -lf 路径
lazy umount是针对busy而提出的，即可以卸载“busy”的文件系统。
请注意，该方法并不是完全安全的

另外可以参考下https://blog.miniasp.com/post/2008/10/20/How-to-unmount-NFS-directory-under-Linux-when-hanging
不过个人很久之前弄得，不记得当时有没用这个方式
