以下摘自维基

  • 资产（アセット，Asset）
    Entrepreneur初入金融街时米达斯银行会赠与一个，是进行战斗的主要媒介。
    型态各异，有人型、动物、人工物等多种外形。拥有角为设计上的共通点。
    据真坂木所言，Asset的外观是Entrepreneur未来的体现。

    • 株（Stock） （华盟是这么翻得,日文是株）:也就是股票
      每一个Asset必定与股票相关、类似于Asset的灵魂，一个Asset有十股股票。
      股票拥有获得Asset的支配权以及股息的授受等机能，是Entrepreneur在金融街生存的重要要素之一。
      拥有强大的能力、或是胜率良好的Entrepreneur所持有的Asset的股价也越高。
      进行Deal时可以卖出股票换取更多的资金，最多可以卖出九股。

  • 交易（Deal）[1]&#91;2]

    金融街的决斗方式，付出成本发出攻击，击中的话就会产生利润，反之，被攻击者被击中的话会损失一定数量的金钱。

    一次交易的时间限制在11分6秒，即666秒。

    一周内最少要进行一次交易(强制性)，最多则无上限。另外，只要投入自己一半的资产给银行就可避过该次的交易。

    • 防御（Guard）

      Asset在未接受Entrepreneur投资的情况下也能使用的能力。能够防御大部份正面攻击，但无法完全防御侧面攻击以及强力（小额以上）的经济魔法。
    • 直接投资（ダイレクト，DIRECT）

      由Entrepreneur本身发出的攻击，外观上为剑的形状，被击中的话会受到庞大的损失。

      不过，发动Direct的Entrepreneur本身也等于暴露在敌方的攻击范围之内，有可能受到敌方的正面攻击，因此Direct可是说是高风险高报酬的投资。

      剑的长度与消耗成本成正比。
    • 经济魔法（Flation）

      Entrepreneur对资产投资后，资产所发动的攻击就称为“经济魔法”。

      经济魔法可以是单纯的破坏魔法，也可以辅助Entrepreneur、迷惑敌人等，拥有各种效果。

      根据投资金额的不同，经济魔法可以分为“小额（Micro）”“中额（Mezzo）”和“巨额（Macro）”三种。
      • 小额（ミクロ，Micro）

        花费以10万元起跳，使用起来最为方便的经济魔法。

        虽然有属性上的不同（例如火与冰），但无论哪个Asset都是物理攻击的经济魔法，可以做出像是“投资100万元，做出10万元5次及50万元1次的攻击”这样细微的指令，或是交由Asset自行判定要投资多少金额（可由资产的角有无发光来判断是否已经入帐），仔细研究的话可做出丰富的战术规划。
      • 中额（メゾ，Mezzo）
        花费以100万元起跳，每个Asset固有的经济魔法。

        从强力的攻击魔法、迷惑对手、反射攻击、预测动向等类型都有，种类非常繁多且不同，通常做为最后一击使用，但也有在交易初期就使用效果会比较显著的中额经济魔法。
        • 焦土作战（Scorched earth）

          真朱的中额经济魔法。弥补真朱耐力不如其他Asset的缺点，非常强力的范围攻击魔法。

          原为军事用语，在商业上指的是被视为并购目标的公司大量出售公司资产，或者破坏公司的特性，以挫败敌意收购人的收购意图。
        • EBO（Employee Buy Out）

          乔治的中额经济魔法。可以制造出大片的烟雾扰乱对手，同时把自己变成佐藤的样子引诱对手攻击。

          原指一个公司中大部份或所有的员工结合在一起，以共同营运（Co-operate）或是运用员工持股计划（Employee Stock Ownership Plan，ESOP）的模式将公司50%以上的股份买下，从而取得所在公司的控制权的行为。
        • 白马骑士（WHITE KNIGHT）

          卡兹的中额经济魔法。能召唤出Entrepreneur所持有的其他Asset，进行双重攻击。

          原指在敌意并购发生时，被视为并购目标公司的友好人士或公司作为第三方出面来解救目标公司、驱逐敌意收购者。
        • 内部交易（Insider）

          格萨的中额经济魔法。在魔法效果发动期间，能够能够完全预测对手的行动。但所需成本过高，需要频繁充值。

          本指在证券市场的交易当中，公司职员及管理者等有权限浏览帐簿的人，利用其职位之便获得未公开的重要内部情报，以此进行相关股票的不正当交易。这种行为在所有国家都是被禁止的。
        • 亡灵扳机（Dead Man’s Trigger）

          贝尔的中额经济魔法。反射型的魔法，身为主人的Entrepreneur受到的伤害越大，“亡灵扳机”的贯穿力与伤害也越强。另外，Entrepreneur也可以自行控制伤害输出。

          原指在遇到收购时，目标公司反过来收购发动袭击的公司。与“反射防御（Pacman Defense）”同义。
        • 自相蚕食（Cannibalization）

          Q的中额经济魔法，发动时会吞噬一定范围的空间，并将踏入此范围的Asset吞噬还原成米达斯货币，增加Entrepreneur的资产总额。

          原指一项新投资会减少本公司已有其他产品或服务的客户、收入或市场份额，或是一些公司（特别是零售连锁公司）在地理位置相近的地方开了两家以上的连锁店。

          通常来说，在对新项目的评估中，其估计的投资收益必须刨除由Cannibalization造成的损失。
      • 巨额（マクロ，Macro）

        花费以1000万元起跳经济魔法，最难以使用的经济魔法。

        比中额更复杂，通常使用了之后效果没办法迅速出现，很难达到收支平衡，但偶尔也有拥有容易使用的巨额魔法的Asset。

        使用后会对asset造成很大的负担。
        • 经济过热（Overheated Economy）

          真朱的巨额经济魔法。发动后从天空召来一道极强大的天火，以火焰覆盖广泛的范围，燃尽一切。

          威力和范围都在“焦土作战（Scorched Earth）”的数倍以上。需要巨额成本并且会对真朱自身造成重负，因此不能连发。

          原指市场供给发展的速度与市场需求发展速度不成比例的现象。当资本增长速度超过市场实际所需要的周期量后，在一定周期阶段内就出现相应的市场资源短缺与一定资源的过剩同时出现的矛盾现象，一定时期其会表现为经济高速发展与物价指数飙高的双高现象。
        • 合并收购（Mergers & Acquisition）

          乔治的巨额经济魔法，在交易中能完全夺取敌人Asset的支配权，但使用次数有限。

          需要花费大量成本，且对乔治自身也会造成重负，因此会限制乔治的行动。

          现实世界中即指企业并购，包括兼并和收购两层含义。企业之间的兼并与收购行为，是企业法人在平等自愿、等价有偿基础上，以一定的经济方式取得其他法人产权的行为，是企业进行资本运作和经营的一种主要形式。国际上习惯将兼并和收购合在一起使用，简称为M&A。
        • 经济封锁（Economic Blockade）

          Q的巨额经济魔法，发动时会冻结敌方Entrepreneur与Asset的一切行动，甚至连已发动的经济魔法都会停止。

          原指封锁贸易及任何买卖交易行为，将一切经济活动限制住的政策。

          通常是经济大国对其他国家的主要限制手段，也是代替战争的有效打击手段之一。
    • 商业模式（Business Model）

      指一个Entrepreneur借由固定的战斗方式持续性地取得Deal的胜利。

      原是指一个完整的产品、服务和信息流体系，包括每一个参与者和其在当中起到的作用，以及每一个参与者的潜在利益和相应的收益来源和方式。简言之，就是公司赚钱的途径和方式。
    • 交易的胜败与报酬

      交易胜败有两种决定方式，分别是“资本增加率”以及“破产”，不同的获胜方式报酬也会不同。

      • 破产：在交易结束之前，迫使对手的总资本额变为0或以下，此时对手在交易中的消费额、剩余的资金以及股份(即 Asset 的所有权)都会归胜者所有。

      • 资本增加率：即“( 后期资本 - 前期资本 ) / 前期资本”在交易时间结束时比较双方的资本增加率，较高的人为胜者，此时获得报酬仅只有对手在交易中的消费额而已。
